lab assignment 3 - vedansh rawat

This project is a simple, responsive portfolio page with sections for About, Skills, Projects, and Contact, aimed at presenting the profile of a budding full stack developer learning the MERN stack. It is ideal as a starter portfolio to share with recruiters, classmates, or collaborators and can be extended into a full MERN application later.​

Features
Clean single-page layout with navigation links to About, Skills, Projects, and Contact sections.​

Brief profile summary focused on teamwork, clean code, and problem-solving mindset.​

Skills section highlighting HTML, CSS, and JavaScript as current core competencies.​
